Utility to apply patches to game ROM-files.

<b>Por favor leia antes de baixar:<b> Este programa não vai te ajudar a hackear jogos nativos para Android, ele foi criado para jogos de consoles antigos (Super Nintendo, Sega MegaDrive, Game Boy Advance, Sony PlayStation etc.).


<b>UniPatcher</b> allows you to apply patches to the game ROM-file. What is a <i>patch</i>? The patch is a file with the modified data of the game. For example, a game translated from Japanese to English. You download the patch containing the translation. It must be applied to the Japanese version to get the game in English.


<b>Recursos:</b>
&bull; Formatos de patch suportados: IPS, IPS32, UPS, BPS, APS (GBA), APS (N64), PPF, DPS, EBP, XDelta3
&bull; Criar patches XDelta
&bull; Consertar checksum em ROMs do Sega Mega Drive
&bull; Removing SMC headers in Super Nintendo ROMs
&bull; Sem anúncios
&bull; Libre software. Source code available at https://github.com/btimofeev/UniPatcher


<b>Como utilizar?</b> Muito simples:
You have to select a ROM file, a patch and choose which file to save, then click on the red round button. Files are selected through the standard Files application (or through one of the file managers you have installed). The application will show a message when the file is patched. Do not close the application until the file is patched.
<b><u>Very important:</u> if the game and patch is compressed (ZIP, RAR, 7z or other), they first need to be unzipped (I recommend the freeware (gratis) program ZArchiver).</b>

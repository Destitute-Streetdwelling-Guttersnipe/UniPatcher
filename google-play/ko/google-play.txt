게임 롬에 패치를 적용하는 유틸리티.

<b>다운로드하기 전에 읽어 주십시오:</b> 이 프로그램은 안드로이드 게임을 해킹하는 데 도움이 되지 않습니다. 오래된 콘솔 게임 (닌텐도 슈퍼 패미컴, 세가 메가 드라이브, 게임보이 어드밴스, 소니 플레이스테이션 등) 용으로 제작되었습니다.


<b>유니패처</b>를 사용하면 게임 롬에 패치를 적용할 수 있습니다. <i>패치</i>란 무엇입니까? 패치는 게임의 수정된 데이터를 포함하고 있는 파일을 말합니다. 예를 들어, 게임이 일본어에서 영어로 번역되었습니다. 당신은 번역된 내용이 들어 있는 패치를 다운로드합니다. 영어 버전을 플레이하려면 일본어 버전에 패치를 적용해야 합니다.


<b>기능:</b>
&bull; 모든 형식의 패치 지원: IPS, IPS32, UPS, BPS, APS (GBA), APS (N64), PPF, DPS, EBP, XDelta3
&bull; XDelta 패치 생성
&bull; 세가 메가 드라이브 롬 체크섬 수정
&bull; Removing SMC headers in Super Nintendo ROMs
&bull; 광고 없음
&bull; 무료 및 오픈소스 소프트웨어. https://github.com/btimofeev/UniPatcher 에서 소스 코드 사용 가능.


<b>어떻게 사용합니까?</b> 매우 간단합니다:
You have to select a ROM file, a patch and choose which file to save, then click on the red round button. Files are selected through the standard Files application (or through one of the file managers you have installed). The application will show a message when the file is patched. Do not close the application until the file is patched.
<b><u>매우 중요:</u> 게임 및 패치가 압축(zip, rar, 7z 등)되어 있는 경우 먼저 압축을 풀어야 합니다(무료 프로그램인 ZArchiver를 권장합니다).</b>


<b>번역을 도와 주십시오:</b>
프로그램을 당신의 언어로 번역하는 데 도움을 줄 수 있습니다. 번역은 https://www.transifex.com/unipatcher/unipatcher 에서 관리됩니다.

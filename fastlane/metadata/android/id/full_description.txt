UniPatcher memungkinkan Anda untuk menerapkan patch ke ROM game.

Apa itu patch?
File dengan data permainan yang dimodifikasi. Misalnya, permainan yang diterjemahkan dari bahasa Jepang ke bahasa Inggris. Anda mengunduh patch yang berisi terjemahan. Ini harus diterapkan pada versi Jepang untuk membuat versi bahasa Inggrisnya.

Program ini tidak akan membantu Anda meretas game Android asli, itu dibuat untuk game konsol lama (Super Nintendo, Sega Mega Drive, Game Boy Advance, Sony PlayStation dll.)

Fitur:
* Format patch yang didukung: IPS, IPS32, UPS, BPS, APS (GBA), APS (N64), PPF, DPS, EBP, XDelta3
* Buat patch XDelta
* Perbaiki checksum di SEGA Mega Drive ROM
* Hapus header SMC dari Super Nintendo ROM
